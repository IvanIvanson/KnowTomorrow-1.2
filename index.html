KnowTomorrow — это приложение для ежедневного отслеживания вашего настроения и окружающей среды. Оценивайте свой день по категориям, анализируйте данные и получайте прогнозы для улучшения качества жизни.

Добро пожаловать в KnowTomorrow! Следуйте этим шагам, чтобы начать использовать приложение:

  1. Выбор категорий: В разделе "Выбор категорий" отметьте категории, которые вы хотите оценивать ежедневно (например, Самочувствие, Погода и т.д.).
  2. Настройка весов: В разделе "Настройка весов категорий" заполните таблицу парного сравнения, чтобы определить важность каждой категории. Сумма весов должна быть равна 1. Нажмите "Сохранить веса".
  3. Оценка дня: В разделе "Оценка дня" выберите дату и оцените выбранные категории по шкале от 1 до 10. Нажмите "Сохранить".
  4. История оценок: Просматривайте свои прошлые оценки в разделе "История оценок". Вы можете удалить записи или добавить оценки за пропущенные дни.
  5. Статистика: Перейдите во вкладку "Статистика" для просмотра графиков:
    • История: Линейный график показывает изменения общей оценки или оценок по категориям.
    • Анализ: Радарный график отображает средние оценки по всем категориям за весь период.
    • Прогноз: На основе ваших данных система предсказывает оценки следующего дня. Нажмите "Сделать прогноз".
  6. Добавить пропущенный день: Используйте модальное окно "Добавить оценку" для внесения оценок за прошлые дни.
